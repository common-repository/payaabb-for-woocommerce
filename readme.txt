== Description ==
PayAABB is the best crypto gateway, which allows users to checkout with popular cryptocurrencies.

How to setup PayAABB on the WooCommerce webstore:
* Download and Activate PayAABB For WooCommerce.
* Enable from WooCommerce > Settings > Payments > PayAABB. 
* Enter API Key from Manage.
* It will automatically appear to the customer on checkout.

= Currencies Support = 
* AABB Gold Token
* Bitcoin
* Bitcoin Cash
* DAI
* Ethereum
* Dogecoin
* USD Coin
* Litecoin
* Tether USD
* ApeCoin
* Shiba Inu.
